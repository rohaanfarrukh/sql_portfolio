# Retail Sales Analysis (MySQL)

## Objective
Analyze retail sales transactions to uncover revenue trends, top pruducts,
regional performance, and average order value using SQL.

## Dataset
Syntheyic retail dataset with 120 transactions across categories, regions 
and dates.

## Key analysis
- Total revenue & AOV
- Revenue by categroy & region
- Top-selling products
- Monthly revenue trend
- High-performing products (Having)

## Tools
- MySQL
- MySQL Workbench
