# SQL Customer Analytics Project

##  Overview
This project demonstrates end-to-end SQL analytics using MySQL.  
It focuses on customer behavior, revenue trends, and retention analysis.

## Tools Used
- MySQL
- MySQL Workbench
- GitHub

##  Dataset
- Customers (500 records)
- Orders (multiple per customer)
- Order items (product-level data)

## Key Analytics Performed
- Customer Lifetime Value (CLV)
- Monthly & rolling revenue trends
- Repeat purchase analysis
- Top 10% high-value customers
- RFM customer segmentation
- Cohort retention analysis
- Product-level revenue analysis

##  Database Schema
Three-table relational schema:
- customers
- orders
- order_items

Primary and foreign keys enforce referential integrity.


## Skills Demonstrated
- SQL joins and aggregations
- Window functions (RANK, NTILE, LAG)
- Common Table Expressions (CTEs)
- Analytical thinking and business insights
- Clean, readable SQL code

## Future Improvements
- Visualization using Power BI
